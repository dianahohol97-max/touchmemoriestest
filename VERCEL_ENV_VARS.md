# Vercel Environment Variables

After adding environment variables to Vercel Dashboard, redeploy is required for them to take effect.

## Current AI Service Variables:
- `GOOGLE_AI_API_KEY` - Google AI API key for AI features
- `NANO_BANANA_API_KEY` - Nano Banana API key for image generation
- `NANO_BANANA_API_ENDPOINT` - Nano Banana API endpoint URL

See `.env` for local development configuration.
