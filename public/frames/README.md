# Frames for PhotoBook Editor

Upload PNG frames with transparent backgrounds here.

## File requirements:
- Format: **PNG** (transparent background)
- Recommended size: **1000x1000px** or larger (square)
- Naming: use descriptive names like `botanical-leaves-01.png`, `gold-ornate-02.png`

## Categories (use prefix in filename):
- `botanical-` — leaves, vines, eucalyptus
- `floral-` — flowers, roses, watercolor flowers
- `gold-` — gold/metallic ornate frames
- `minimal-` — thin lines, geometric
- `vintage-` — classic, antique style
- `wedding-` — hearts, ribbons, elegant
- `kids-` — stars, clouds, rainbow
- `lineart-` — line art illustrations

## How to upload:
1. Go to this folder on GitHub
2. Click "Add file" → "Upload files"
3. Drag and drop your PNG files
4. Commit

The constructor will automatically pick up new frames after deploy.
