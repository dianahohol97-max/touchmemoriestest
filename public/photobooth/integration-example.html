📸 Online Photobooth - Integration Example

How to Integrate

To integrate the photobooth into your website, add an iframe to your page:

<!-- Basic Integration -->
<iframe
  src="https://yourdomain.com/photobooth"
  width="100%"
  height="600px"
  frameborder="0"
  allow="camera; microphone"
></iframe>

Required Permissions:

  • allow="camera" - Required for camera access
  • HTTPS is required for camera permissions to work

React/Next.js Integration

import { PhotoboothEmbed } from '@/components/photobooth';

function MyPage() {
  return (
    <PhotoboothEmbed
      initialConfig={{
        capture: { numberOfPhotos: 4 },
        customization: { eventName: 'My Event' }
      }}
      allowConfiguration={true}
    />
  );
}

Configuration Options

You can customize the photobooth by passing URL parameters:

https://yourdomain.com/photobooth?layout=photostrip_2x6&photos=4&event=Wedding

Available Parameters:

  • layout - photostrip_2x6, print_4x6_grid, square_instagram, etc.
  • photos - Number of photos to capture (1-10)
  • countdown - Countdown seconds (1-10)
  • event - Event name to display on photos
  • camera - user (front) or environment (back)
