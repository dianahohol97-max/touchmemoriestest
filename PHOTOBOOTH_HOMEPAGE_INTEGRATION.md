# 📸 Інтеграція Фотобудки на Головну Сторінку

## Що було додано

На головну сторінку сайту Touch Memories було додано нову секцію **"Онлайн Фотобудка"** перед секцією замовлення тревел-буку.

---

## 🎯 Розташування

Секція фотобудки з'являється після секції "How It Works" і перед секцією "Travel":

```
Головна сторінка (/)
├── Hero
├── Featured Products
├── Categories
├── How It Works
├── 📸 PHOTOBOOTH SECTION ← НОВА СЕКЦІЯ
├── Travel Section
├── Price Calculator
└── ...інші секції
```

**defaultOrder:** 4.5 (між 4 та 5)

---

## 📁 Створені файли

### 1. PhotoboothSection Component
**Файл:** `components/ui/PhotoboothSection.tsx`

Головний компонент секції, який включає:
- Привабливий заголовок та опис
- 4 ключові переваги з іконками
- Кнопку запуску фотобудки
- Візуальний макет телефону з прев'ю
- Анімовані елементи (✨🎉💫)
- Сітку з доступними форматами друку
- Повноекранний режим фотобудки

### 2. PhotoboothSection Styles
**Файл:** `components/ui/PhotoboothSection.module.css`

Стилі включають:
- Градієнтний фон (purple-to-violet)
- Адаптивний дизайн (mobile-first)
- Анімації (float, hover effects)
- Макет телефону з прев'ю
- Сітка форматів друку

## 🎨 Дизайн Секції

### Кольорова Схема
- **Фон:** Градієнт від #667eea до #764ba2
- **Текст:** Білий
- **Акценти:** Напівпрозорі білі елементи
- **Кнопка:** Біла з фіолетовим текстом

### Візуальні Елементи

1. **Текстовий Контент (Ліворуч)**
   - Бейдж "📸 Новинка"
   - Заголовок "Онлайн Фотобудка"
   - Опис функціоналу
   - 4 переваги в сітці 2×2
   - Кнопка "Спробувати фотобудку"
   - Підказка про використання

2. **Візуальний Макет (Праворуч)**
   - 3D макет телефону
   - Прев'ю фото-макету всередині
   - Плаваючі емоджі (✨🎉💫)

3. **Формати Друку (Знизу)**
   - 4 картки з прев'ю форматів:
     - Фотострічка 2×6" (3 фото)
     - Сітка 4×6" (4 фото)
     - Квадрат 4×4" (4 фото)
     - Альбомна 6×4" (2 фото)

## ⚡ Функціонал

### Інтерактивність

1. **Кнопка "Спробувати фотобудку"**
   - Відкриває повноекранну фотобудку
   - Анімація при наведенні
   - Стрілка рухається вправо

2. **Повноекранний Режим**
   - Фотобудка займає весь екран
   - Кнопка закриття (✕) у верхньому правому куті
   - Інтеграція з PhotoboothEmbed компонентом

3. **Налаштування Фотобудки**
   ```tsx
   layout: LAYOUTS.photostrip_2x6
   numberOfPhotos: 3
   countdownSeconds: 3
   delayBetweenShots: 1000
   cameraFacing: 'user'
   eventName: 'Touch Memories'
   textColor: '#FFD700'
   ```

## 📱 Адаптивність

### Desktop (> 968px)
- Сітка 2 колонки (текст | візуал)
- Формати друку: 4 колонки

### Tablet (640px - 968px)
- Сітка 1 колонка
- Візуал зверху
- Формати друку: 2 колонки

### Mobile (< 640px)
- Все в 1 колонку
- Зменшений розмір телефону
- Формати друку: 1 колонка (горизонтальні картки)
- Кнопка на всю ширину

## 🔧 Технічні Деталі

### Імпорт на Головній Сторінці

```tsx
import { PhotoboothSection } from '@/components/ui/PhotoboothSection';

// В JSX:
<SectionWrapper name="photobooth" defaultOrder={4.5}>
  <PhotoboothSection />
</SectionWrapper>
```

### Стан Компонента

```tsx
const [showPhotobooth, setShowPhotobooth] = useState(false);
```

- `false` - показує промо-секцію
- `true` - показує повноекранну фотобудку

### Callbacks

```tsx
const handleComplete = (imageDataUrl: string) => {
  console.log('Фото готове!');
  // Можна додати збереження на сервер
};

const handleError = (error) => {
  console.error('Помилка:', error);
  alert(error.message);
};
```

## 🐛 Troubleshooting

### Фотобудка не відкривається
✅ Перевірте, чи імпортовано PhotoboothSection
✅ Перевірте console на помилки
✅ Переконайтеся, що HTTPS увімкнено

### Стилі не застосовуються
✅ Перевірте, чи CSS модуль імпортовано
✅ Очистіть кеш Next.js (.next папка)
✅ Перезапустіть dev server

### Камера не працює
✅ Дозвольте дозволи в браузері
✅ Використовуйте HTTPS
✅ Перевірте підтримку WebRTC
